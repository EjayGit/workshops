# Design Principles

## 10 principles:
- Type of Site: what is its purpose, layout, wording.
- User Flow: Steer towards closure.
- Hierarchy: Define the importance on info/items. Decisiveness.
- Consistency: Keeping font, colour schemes the same. Reuse components.
- Balance: white space is your friend, increases user focus.
- Simplicity: Clear and simple for user. Reduce options. Reduce cognitive load.
- Contrast: Think ease of reading, think different screens and colourblindness. 
- Imagery: emotional resonance, 
- Typography: keep it simple, readable, consistent, dyslexia friendly.
- User Feedback: symbolic reactions, audio, and messages can help users understand that their intent has been successful or not. Give it interactivity with the user. hover, focus, active, disable etc.
